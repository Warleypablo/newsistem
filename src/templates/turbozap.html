TurboZap

Sistema automatizado de envio de mensagens WhatsApp para gestão de cobranças e comunicação com clientes inadimplentes.

Funcionalidades

O TurboZap oferece um conjunto completo de ferramentas para automatizar a comunicação via WhatsApp:

  • Envio automatizado de mensagens
  • Templates personalizados por período
  • Gestão de clientes inadimplentes
  • Integração com Evolution API
  • Relatórios detalhados de envio
  • Interface de linha de comando
Documentação

Como Usar

Configure e execute o TurboZap em poucos passos:

  • Configure as variáveis de ambiente
  • Conecte com sua instância Evolution API
  • Defina os períodos de cobrança
  • Execute via linha de comando
  • Monitore os envios em tempo real
